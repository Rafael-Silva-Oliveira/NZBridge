Latest releases and upcoming features.

v0.2.1 Current

  • Fix: PDF uploads no longer silently drop files when syncing large collections — all PDFs now upload reliably
  • Up to 3 automatic retries per file with backoff if the upload dialog doesn't respond in time
  • Better error messages — shows exactly which PDFs failed and how many, instead of truncating to 3 errors
  • Note: upload time per file may be slightly longer than v0.2.0 to ensure reliability. Actual time depends on PDF size, your internet connection speed, and NotebookLM's response time

v0.2.0

  • Selective sync — choose exactly which PDFs and URLs to upload before syncing
  • Real-time progress bar with per-file status and item counter
  • Progress is preserved if the popup is closed and reopened mid-sync
  • Faster PDF uploads — ~3-5s per file (down from ~10-15s)
  • Fixed: PDFs no longer open as new browser tabs instead of uploading

v0.1.3

  • The 50-source limit is now a warning, not a blocker — Pro plan users can sync collections larger than 50 items

v0.1.2

  • Fix: correct NotebookLM tab targeting when multiple notebook tabs are open
  • Fix: "Add sources" failures caused by syncing into the wrong tab
  • Fix: file-chooser interception now always cleaned up on error (restores manual uploads)
  • Fix: PDF uploads on macOS and Linux (path separator handling)
  • Fix: clearer errors when Zotero is not running or the plugin is unavailable
  • Fix: popup no longer hangs forever if the background worker is slow
  • Guard: prevents two syncs from running simultaneously on the same tab
  • Backward sync prefers the originally mapped notebook when multiple are open

v0.1.0

  • Forward sync: upload PDFs and URLs from Zotero to NotebookLM
  • Backward sync: extract text notes from NotebookLM Studio
  • Smart URL selection (prefers open-access over bot-protected links)
  • Auto-naming of new notebooks after Zotero collections
  • Parent document items for Notion/Notero compatibility
  • Custom tags and default tags on imported notes
  • Collapsed/vertical layout support
  • Duplicate detection and overwrite on re-import

Planned Upcoming

  • Import audio overview transcripts
  • Import mind maps and other Studio formats
  • Selective source sync (choose which items to push)
